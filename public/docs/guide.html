1. Tổng quan

USDT Gateway là hệ thống cổng thanh toán USDT (TRC20) tự động, cho phép đối tác (partner) tích hợp nạp/rút USDT thông qua REST API.

3. Admin CMS

Truy cập: /admin/ — Đăng nhập bằng tài khoản admin.

📊 Dashboard

  • Tổng nạp/rút USDT (hôm nay + tổng cộng)
  • Số dư ví on-chain real-time (truy vấn trực tiếp blockchain)
  • Tỷ giá USDT/VND hiện tại

📥 Tab Nạp USDT

  • Danh sách tất cả lệnh nạp với phân trang (10/trang)
  • Tìm kiếm theo mã đơn, TX Hash, địa chỉ ví
  • Lọc theo ví nạp cụ thể
  • Xác nhận thủ công cho giao dịch đặc biệt
  • Bảng "Giao dịch không khớp" — TX nhận được nhưng không match đơn nào

📤 Tab Rút USDT

  • Danh sách lệnh rút: Chờ duyệt / Đã duyệt / Đang gửi / Đã gửi / Thất bại / Từ chối
  • Duyệt / Từ chối lệnh rút
  • 2FA OTP qua Telegram (nếu bật trong cài đặt)

💼 Tab Ví

  • Quản lý ví nạp/rút riêng biệt
  • Thêm ví mới (địa chỉ + private key tự mã hóa)
  • Import từ seed phrase 12 từ → tự derive private key
  • Số dư USDT + TRX (gas) real-time từ blockchain
  • Bật/tắt/xóa ví

👥 Tab Đối tác

  • Tạo partner: tên, API key (tự sinh), secret key, callback URL
  • Cấu hình spread (chênh lệch giá mua/bán) cho từng partner

6. Bảo mật

  • Private Key mã hóa AES-256-CBC trong database
  • API Key xác thực mỗi partner
  • HMAC-SHA256 Signature cho webhook callback
  • 2FA OTP (Telegram) cho duyệt rút — bật/tắt trong cài đặt
  • Rate Limiting API — cấu hình số request/phút
  • IP Whitelist cho admin — bật/tắt + danh sách IP
  • Giới hạn rút/ngày + Cooldown chống rút ồ ạt
  • Login Alert — mỗi lần admin đăng nhập đều thông báo Telegram (IP, thiết bị)

7. Flow hoạt động

📥 Flow Nạp USDT

Partner gọi POST /deposit
→ Hệ thống tạo đơn + gán ví nhận
→ Partner hiển thị QR cho user
→ User chuyển USDT vào ví (đúng số tiền)
→ Cron quét blockchain phát hiện TX
→ Khớp số tiền → trạng thái CONFIRMED
→ Webhook callback về partner
→ Telegram alert ✅

📤 Flow Rút USDT

Partner gọi POST /withdraw
→ Hệ thống tạo đơn PENDING
→ [Auto mode] Dưới ngưỡng → tự động APPROVED
→ [Manual mode] Admin duyệt (kèm OTP nếu 2FA bật)
→ Cron gửi USDT on-chain → SENT
→ Webhook callback về partner
→ Telegram alert 💸

8. Trạng thái giao dịch

Nạp:

PENDING → CONFIRMED / EXPIRED

Rút:

PENDING → APPROVED → SENDING → SENT / FAILED / REJECTED

9. Lưu ý quan trọng

⚠️ Hệ thống chạy trên TRON Mainnet — mọi giao dịch đều dùng USDT thật!
  1. Khi nạp, user phải chuyển đúng số USDT đã tạo đơn. Sai số tiền → không tự động xác nhận.
  2. Khi rút, kiểm tra kỹ địa chỉ ví nhận. Sai ví → USDT mất không lấy lại được.
  3. Ví rút cần ≥ 30 TRX để trả phí gas. Hết TRX → lệnh rút thất bại.
  4. Không chia sẻ API Key / Secret Key / Private Key ra ngoài.
  5. ENCRYPTION_KEY không được thay đổi sau khi đã có ví trong hệ thống.
  6. Backup database PostgreSQL định kỳ.
